Absolute Cinema Frontend

Frontend of Absolute Cinema, a movie recommendation web application built using React.

This project allows users to create an account, log in, select a movie genre, and get movie recommendations generated by the backend recommendation system. Recommendation history is stored for each user.

Features
User Signup
User Login
JWT Authentication
Genre Based Recommendations
Recommendation History
User Profile Page
Built With
React
React Router
JavaScript
CSS
Fetch API
React Icons
Running the Project

Install dependencies:

npm install

Start the development server:

npm run dev

Frontend runs on:

http://localhost:5173
Backend

This frontend works with the Absolute Cinema Spring Boot backend.

Backend Repository:
https://github.com/4n4ndd/AbsoluteCinemaBackend

Make sure the backend server is running before starting the application.
